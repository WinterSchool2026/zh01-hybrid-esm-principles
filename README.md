# Hybrid modeling principles and concepts for earth system models

## Description

In this session, participants explore why classical parameterizations in Earth system models—such as eddy-viscosity closures in coarse-resolution turbulence simulations—systematically fail to capture key dynamical features. Using a 2D turbulence example, we examine over-damping effects, loss of coherent vortices, and incorrect energy spectra, motivating the need for hybrid and learned parameterizations. The tutorial provides a conceptual foundation for understanding how AI can complement physical modeling in next-generation Earth system models.
